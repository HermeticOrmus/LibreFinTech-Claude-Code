# Beginner Learning Path - Financial Technology

## Overview

This beginner path covers Financial Technology concepts and practices, progressing from fundamentals to practical application with Claude Code plugins.

## Prerequisites

- Programming fundamentals\n- Basic understanding of finance\n- Database knowledge (SQL)

## Modules

### Module 1: FinTech Fundamentals

#### Concepts
- Financial system overview\n- Payment processing basics\n- Double-entry bookkeeping\n- Regulatory landscape overview\n- Financial data types and precision

#### Hands-On Exercise
Apply the concepts using the relevant plugins. Start simple, build complexity gradually.

#### Key Takeaways
- Understand the fundamentals of FinTech Fundamentals
- Know when to apply these patterns
- Create basic implementations confidently

---

### Module 2: Ledger Design

#### Concepts
- Account structures\n- Transaction modeling\n- Balance calculation strategies\n- Idempotency in financial operations\n- Audit trail fundamentals

#### Hands-On Exercise
Build on Module 1 with more advanced techniques. Focus on quality and reproducibility.

#### Key Takeaways
- Integrate multiple concepts effectively
- Apply established best practices
- Evaluate and improve existing work

---

### Module 3: Payment Basics

#### Concepts
- Payment flow lifecycle\n- Card processing basics\n- Payment gateway integration\n- Error handling in payments\n- Refunds and chargebacks

#### Hands-On Exercise
Combine everything into a comprehensive project demonstrating full mastery.

#### Key Takeaways
- Synthesize all module concepts
- Make informed architectural decisions
- Create production-quality implementations

## Assessment

- [ ] Complete all module exercises

Continue to **intermediate** for KYC/AML, risk scoring, and API integration.
