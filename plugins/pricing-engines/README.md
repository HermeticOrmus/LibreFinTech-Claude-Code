# Pricing Engines

Dynamic pricing, fee calculation, interest computation

## What's Included

### Agents
- **Pricing Engineer** - Specialized agent for Dynamic pricing, fee calculation, interest computation

### Commands
- `/pricing` - Quick-access command for pricing-engines workflows

### Skills
- **Pricing Patterns** - Pattern library and knowledge base for pricing-engines

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/pricing analyze

# Use the command with specific input
/pricing generate --context "your project"

# Reference patterns from the skill
"Apply pricing-patterns patterns to this implementation"
```
