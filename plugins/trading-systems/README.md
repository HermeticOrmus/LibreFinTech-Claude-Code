# Trading Systems

Order management, matching engines, FIX protocol

## What's Included

### Agents
- **Trading Engineer** - Specialized agent for Order management, matching engines, FIX protocol

### Commands
- `/trading` - Quick-access command for trading-systems workflows

### Skills
- **Trading Patterns** - Pattern library and knowledge base for trading-systems

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/trading analyze

# Use the command with specific input
/trading generate --context "your project"

# Reference patterns from the skill
"Apply trading-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for trading-systems
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
