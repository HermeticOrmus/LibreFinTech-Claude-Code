# Lending Platforms

Loan origination, credit scoring, servicing, collections

## What's Included

### Agents
- **Lending Engineer** - Specialized agent for Loan origination, credit scoring, servicing, collections

### Commands
- `/lending` - Quick-access command for lending-platforms workflows

### Skills
- **Lending Patterns** - Pattern library and knowledge base for lending-platforms

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/lending analyze

# Use the command with specific input
/lending generate --context "your project"

# Reference patterns from the skill
"Apply lending-patterns patterns to this implementation"
```
