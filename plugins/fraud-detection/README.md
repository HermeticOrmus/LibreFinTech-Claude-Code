# Fraud Detection

Rule engines, ML fraud detection, anomaly scoring

## What's Included

### Agents
- **Fraud Engineer** - Specialized agent for Rule engines, ML fraud detection, anomaly scoring

### Commands
- `/fraud-detect` - Quick-access command for fraud-detection workflows

### Skills
- **Fraud Detection Patterns** - Pattern library and knowledge base for fraud-detection

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/fraud-detect analyze

# Use the command with specific input
/fraud-detect generate --context "your project"

# Reference patterns from the skill
"Apply fraud-detection-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for fraud-detection
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
