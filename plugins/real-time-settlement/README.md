# Real Time Settlement

RTGS, instant payments, clearing, netting

## What's Included

### Agents
- **Settlement Engineer** - Specialized agent for RTGS, instant payments, clearing, netting

### Commands
- `/settlement` - Quick-access command for real-time-settlement workflows

### Skills
- **Settlement Patterns** - Pattern library and knowledge base for real-time-settlement

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/settlement analyze

# Use the command with specific input
/settlement generate --context "your project"

# Reference patterns from the skill
"Apply settlement-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for real-time-settlement
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
