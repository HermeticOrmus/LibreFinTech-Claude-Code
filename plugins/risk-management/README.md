# Risk Management

Market risk, credit risk, operational risk, VaR, stress testing

## What's Included

### Agents
- **Risk Engineer** - Specialized agent for Market risk, credit risk, operational risk, VaR, stress testing

### Commands
- `/risk` - Quick-access command for risk-management workflows

### Skills
- **Risk Patterns** - Pattern library and knowledge base for risk-management

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/risk analyze

# Use the command with specific input
/risk generate --context "your project"

# Reference patterns from the skill
"Apply risk-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for risk-management
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
