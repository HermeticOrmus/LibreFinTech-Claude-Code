# Financial Security

PCI DSS, encryption, tokenization, secure transactions

## What's Included

### Agents
- **Finsec Engineer** - Specialized agent for PCI DSS, encryption, tokenization, secure transactions

### Commands
- `/fin-security` - Quick-access command for financial-security workflows

### Skills
- **Fin Security Patterns** - Pattern library and knowledge base for financial-security

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/fin-security analyze

# Use the command with specific input
/fin-security generate --context "your project"

# Reference patterns from the skill
"Apply fin-security-patterns patterns to this implementation"
```

- Follow established conventions for financial-security
