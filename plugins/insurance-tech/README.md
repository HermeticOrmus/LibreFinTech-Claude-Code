# Insurance Tech

Claims processing, underwriting, policy management

## What's Included

### Agents
- **Insurtech Engineer** - Specialized agent for Claims processing, underwriting, policy management

### Commands
- `/insurtech` - Quick-access command for insurance-tech workflows

### Skills
- **Insurtech Patterns** - Pattern library and knowledge base for insurance-tech

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/insurtech analyze

# Use the command with specific input
/insurtech generate --context "your project"

# Reference patterns from the skill
"Apply insurtech-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for insurance-tech
- Validate inputs before processing
- Document decisions and rationale
