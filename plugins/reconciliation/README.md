# Reconciliation

Transaction reconciliation, matching algorithms, exceptions

## What's Included

### Agents
- **Reconciliation Engineer** - Specialized agent for Transaction reconciliation, matching algorithms, exceptions

### Commands
- `/reconcile` - Quick-access command for reconciliation workflows

### Skills
- **Reconciliation Patterns** - Pattern library and knowledge base for reconciliation

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/reconcile analyze

# Use the command with specific input
/reconcile generate --context "your project"

# Reference patterns from the skill
"Apply reconciliation-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for reconciliation
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
