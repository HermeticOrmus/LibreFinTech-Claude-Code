# Open Banking

Open banking standards, consent management, API security

## What's Included

### Agents
- **Open Banking Engineer** - Specialized agent for Open banking standards, consent management, API security

### Commands
- `/open-banking` - Quick-access command for open-banking workflows

### Skills
- **Open Banking Patterns** - Pattern library and knowledge base for open-banking

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/open-banking analyze

# Use the command with specific input
/open-banking generate --context "your project"

# Reference patterns from the skill
"Apply open-banking-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for open-banking
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
