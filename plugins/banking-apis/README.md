# Banking Apis

Open banking APIs, PSD2, account aggregation

## What's Included

### Agents
- **Banking Api Engineer** - Specialized agent for Open banking APIs, PSD2, account aggregation

### Commands
- `/banking-api` - Quick-access command for banking-apis workflows

### Skills
- **Banking Api Patterns** - Pattern library and knowledge base for banking-apis

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/banking-api analyze

# Use the command with specific input
/banking-api generate --context "your project"

# Reference patterns from the skill
"Apply banking-api-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for banking-apis
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
