# Regulatory Compliance

SOX, MiFID II, Dodd-Frank, regulatory reporting

## What's Included

### Agents
- **Compliance Engineer** - Specialized agent for SOX, MiFID II, Dodd-Frank, regulatory reporting

### Commands
- `/compliance` - Quick-access command for regulatory-compliance workflows

### Skills
- **Compliance Patterns** - Pattern library and knowledge base for regulatory-compliance

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/compliance analyze

# Use the command with specific input
/compliance generate --context "your project"

# Reference patterns from the skill
"Apply compliance-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for regulatory-compliance
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
