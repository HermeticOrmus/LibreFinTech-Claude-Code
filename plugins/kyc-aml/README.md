# Kyc Aml

Identity verification, sanctions screening, transaction monitoring

## What's Included

### Agents
- **Kyc Aml Engineer** - Specialized agent for Identity verification, sanctions screening, transaction monitoring

### Commands
- `/kyc-aml` - Quick-access command for kyc-aml workflows

### Skills
- **Kyc Aml Patterns** - Pattern library and knowledge base for kyc-aml

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/kyc-aml analyze

# Use the command with specific input
/kyc-aml generate --context "your project"

# Reference patterns from the skill
"Apply kyc-aml-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for kyc-aml
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
