# Portfolio Management

Portfolio construction, rebalancing, asset allocation

## What's Included

### Agents
- **Portfolio Engineer** - Specialized agent for Portfolio construction, rebalancing, asset allocation

### Commands
- `/portfolio` - Quick-access command for portfolio-management workflows

### Skills
- **Portfolio Patterns** - Pattern library and knowledge base for portfolio-management

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/portfolio analyze

# Use the command with specific input
/portfolio generate --context "your project"

# Reference patterns from the skill
"Apply portfolio-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for portfolio-management
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
