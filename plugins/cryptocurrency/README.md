# Cryptocurrency

Blockchain integration, wallet management, smart contracts

## What's Included

### Agents
- **Crypto Engineer** - Specialized agent for Blockchain integration, wallet management, smart contracts

### Commands
- `/crypto` - Quick-access command for cryptocurrency workflows

### Skills
- **Crypto Patterns** - Pattern library and knowledge base for cryptocurrency

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/crypto analyze

# Use the command with specific input
/crypto generate --context "your project"

# Reference patterns from the skill
"Apply crypto-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for cryptocurrency
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
