# Payment Processing

Payment gateways, Stripe/Adyen, 3DS, recurring billing

## What's Included

### Agents
- **Payments Engineer** - Specialized agent for Payment gateways, Stripe/Adyen, 3DS, recurring billing

### Commands
- `/payments` - Quick-access command for payment-processing workflows

### Skills
- **Payment Patterns** - Pattern library and knowledge base for payment-processing

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/payments analyze

# Use the command with specific input
/payments generate --context "your project"

# Reference patterns from the skill
"Apply payment-patterns patterns to this implementation"
```

- Follow established conventions for payment-processing
