# Financial Reporting

Financial statements, GAAP/IFRS, automated reporting

## What's Included

### Agents
- **Financial Reporter** - Specialized agent for Financial statements, GAAP/IFRS, automated reporting

### Commands
- `/fin-report` - Quick-access command for financial-reporting workflows

### Skills
- **Fin Reporting Patterns** - Pattern library and knowledge base for financial-reporting

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/fin-report analyze

# Use the command with specific input
/fin-report generate --context "your project"

# Reference patterns from the skill
"Apply fin-reporting-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for financial-reporting
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
