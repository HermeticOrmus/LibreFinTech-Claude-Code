# Market Data

Market data feeds, tick data, OHLCV, normalization

## What's Included

### Agents
- **Market Data Engineer** - Specialized agent for Market data feeds, tick data, OHLCV, normalization

### Commands
- `/market-data` - Quick-access command for market-data workflows

### Skills
- **Market Data Patterns** - Pattern library and knowledge base for market-data

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/market-data analyze

# Use the command with specific input
/market-data generate --context "your project"

# Reference patterns from the skill
"Apply market-data-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for market-data
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
