# Audit Trails

Immutable audit logs, event sourcing for finance, compliance

## What's Included

### Agents
- **Audit Engineer** - Specialized agent for Immutable audit logs, event sourcing for finance, compliance

### Commands
- `/audit-trail` - Quick-access command for audit-trails workflows

### Skills
- **Audit Trail Patterns** - Pattern library and knowledge base for audit-trails

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/audit-trail analyze

# Use the command with specific input
/audit-trail generate --context "your project"

# Reference patterns from the skill
"Apply audit-trail-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for audit-trails
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
